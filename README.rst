========================================================
サクラエディタの RestructuredText 向けのタイプ別設定です
========================================================

使用方法
========

設定 > タイプ別設定一覧 > インポート を選択し、ファイルダイアログで reST.ini を選択してください。

環境
=====

サクラエディタ ver 2.0.4.0 と、breonig.dll ver. 3.0.1 の組み合わせで動作確認しています。
